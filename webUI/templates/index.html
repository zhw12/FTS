FTS: A Bird’s Eye View Of Science & Technology

Driven by AI


FTS, Faceted Taxonomy Construction and Search, provides a bird’s eye view of scientific papers and technical reports, making technology exploration much easier and faster. FTS leverages state-of-the-art network science, data mining and deep learning techniques to automatically mine concepts and construct concept networks like taxonomy from publication repository. The discovered concepts and their networks could significantly simplify decision making.

Major Functions

    Search

    FTS Search provides intelligent search functionality enhanced with query suggestion and taxonomy navigation.

    Taxonomy Exploration

    FTS Taxonomy Exploration allows user to explore the generated taxonomies.

    Taxonomy Generation

    FTS Taxonomy Generation allows user to generate a taxonomy by assigning documents by query.

    Area Analysis

    FTS Area Analysis visualizes the research strength in different areas under taxonomy.

    Trend Analysis

    FTS Trend Analysis provides trend analysis for different research areas across years.

    Trend Analysis

    FTS Trend Analysis provides trend analysis for different countries across years.

    Author Cloud

    FTS Author Cloud provides function for finding major players in an area.

    Concept Cloud

    FTS Concept Cloud provides function for finding influential topics in an area.
